Pricipais Formatações

Negrito ou Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica)

Nesta ou frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica)

Nesta frase temos um termo em destaque usando a tag EM (semântica)

Texto Marcado

Podemos também criar um texto marcado usando a tag MARK

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno com as tags BIG (não semântica) e SMALL(semântica).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas nao considerado, com a tag DEL.(semântica)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase de que ele foi adicionado depois, com a tag INS(semântica)

Também podemos sublinhar um texto com a tag U (não semântica).

Texto sobrescrito

Para inserir coisas do tipo x20+3, com a tag SUP.

Texto subscrito

Para inserir coisas do tipo H2O, com a tag SUB.
